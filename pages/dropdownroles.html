Dropdown Roles

About the bot

Dropdown roles is a bot utilizing the latest interaction type on discord: dropdowns!

Users just have to run /dropdownroles use to send a dropdown which allows them to select up to 25 roles to add or remove from them.
If a user already has the role they select, it will be removed from them

You can invite the bot here


User Commands

/dropdownroles use this command is main command for the bot. It sends a dropdown with all the roles the server admins have configured (see admin commands)

/stats sends a simple embed containg some stats about the bot's overall usage

/info sends an embed similar to /stats, however this embed contains general info on the bot, and a command reference


Admin Commands

All admin commands require the manage roles permission

/dropdownroles add This is a command that adds a given role to all future dropdowns sent.

Please note: you cannot add a role that is higher than the bot's integration role, or a role that is managed by another bot

/dropdownroles remove This command removes a role from all future dropdowns sent.

/dropdownroles send This command sends a button, that when clicked, will send an ephemeral dropdown to the person who clicked it
